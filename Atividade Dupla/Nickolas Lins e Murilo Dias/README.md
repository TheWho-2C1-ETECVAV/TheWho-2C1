# 📘 Lista de Exercícios em PHP

## 📌 Sobre o projeto

Este repositório contém a resolução de uma lista de exercícios em PHP, com foco no desenvolvimento da lógica de programação.

Os exercícios abordam conceitos fundamentais como:

* Estruturas condicionais
* Laços de repetição
* Funções
* Manipulação de arrays

---

## 🧩 Estrutura do projeto

Cada exercício está organizado em uma pasta separada:

```
ex01/
ex02/
ex03/
...
```

Dentro de cada pasta:

* `index.php` → interface / entrada de dados
* `calcula.php` → processamento da lógica

---

## 🎯 Objetivo

Praticar e consolidar conceitos básicos de programação utilizando PHP, com foco em organização de código e separação de responsabilidades.
